About Me

I'm a Computer Science and Applied Mathematics student at UC Berkeley with a concentration in Economics. Exploring the intersections of mathematics, computer science, and whatever else life throws my way.

Currently exploring the fascinating world of expander graphs and group theory through Berkeley's Mathematics DRP. Quick disclaimer: in this process, I have discovered that front-end development is not my forte. But hey, we all have our strengths.

When I'm not buried in math theorems or debugging code, you'll find me performing with Cal Bhangra dance team or working as a Sutardja Dai Center Innovation Fellow. I'm drawn to unexpected connections between disciplines — sometimes the best solutions come from places you wouldn't expect.

Expander Graph Visualization Tool

January 2025 Personal Project

Developed an interactive visualization tool for exploring expander graphs and their properties. Built with Python and matplotlib, with a focus on visualizing spectral properties and connectivity.

Python NetworkX Graph Theory Data Visualization

Bayesian Decision Model

October 2023 Research Project

Built a computational model of decision-making under uncertainty using Bayesian methods. Implemented in Python with PyMC for MCMC sampling and parameter estimation. Model incorporated prior beliefs, utility functions, and learning mechanisms.

Python PyMC Bayesian Statistics Decision Theory

CubeSat Orientation System

2020-2022 Irvine CubeSat

Designed and implemented a star tracking system for CubeSat orientation using OpenCV and public star databases. Integrated with IMU sensor data for more robust orientation estimates.
